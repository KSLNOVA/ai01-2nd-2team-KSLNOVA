## 실행 가이드 (로컬, 번들러 없이)

이 프로젝트는 React 코드를 CDN + Babel로 구동합니다. Node 없이도 브라우저에서 바로 열 수 있지만, API 키를 담은 `env.js`를 생성해야 합니다.

### 1) .env 준비
- 루트 `.env` 파일에 다음 키를 넣어둡니다.
  ```
  YOUTUBE_API_KEY=...   # 선택: YouTube 검색용
  OPENAI_API_KEY=...    # 선택: LLM/TTS 피드백용
  ```

### 2) env.js 생성 (키를 브라우저에 주입)
- 루트에서 실행:
  ```
  python generate_env_js.py
  ```
- `env.js`가 생성됩니다. **Git에 올리지 마세요**(`.gitignore`에 포함되어 있어야 함).

### 3) 로컬 서버로 열기
- 루트에서 간단한 HTTP 서버 실행:
  ```
  python -m http.server 8000
  ```
- 브라우저에서 `http://localhost:8000/index.html` 접속.
  - YouTube/LLM 기능은 외부 네트워크가 열려 있어야 합니다.
  - 브라우저에서 카메라 접근 권한을 허용해야 실시간 영상이 보입니다.

### 4) 주요 기능
- 운동 선택 → YouTube 추천 + 지난 기록 갱신.
- 웹캠 스트림 → 자세 피드백 표시(TTS 버튼으로 음성 안내).
- LLM 챗: OpenAI 키가 있으면 답변, 없으면 안내 메시지.

### 5) 주의
- `env.js`에는 실 키가 들어가므로 **절대 커밋/푸시하지 마세요**.
- 공개 배포 시에는 백엔드 프록시를 두어 키를 숨기거나, 브라우저 키 주입 방식을 사용하지 마세요.
